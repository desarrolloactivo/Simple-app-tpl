Como configurar nuestro servidor VPS para que trabaje junto a GitHub y lograr una buena experiencia de trabajo colaborativo.




1) Configurar ssh key entre github y el vps

https://help.github.com/articles/generating-ssh-keys

2) Despues de realizar la configuracion anterior y tener nuestro repositorio creado en github vamos a conectarnos a nuestro servidor VPS para copiar el repositorio desde nuestro GitHub.

Una vez conectado por medio de ssh en el servidor vps nos dirigimos a la carpeta donde vamos a crear nuestro repocitorio

Ejemplo:

usuario/public_html/test/

Una vez dentro del directorio vamos a clonar nuestro repositorio github por medio del siguiente comando:

git --init git clone git@github.com:NombreDeUsuarui/NombreDeRepositorio.git

y con esto se debe crear una carpeta dentro del directorio actual con el nombre de los repositorios

luego nos movemos a la carpeta de nuestro repocitorio recien creado (segun nuestro ejemplo seria NombreDeRepocitorio)

cd /public_html/test/NombreDeRepositorio

Lo siguiente seria hacer un test manual realizando cambios en el repocitorio y luego mediante

git pull

Revisamos si se actualizaron los archivos del repocitorio en github a nuestro VPS

3) Creamos un archivo php Ejemplo git.php

Dentro de este archivo vamos a agregar el siguiente codigo php



< ? php `git pull` ;


Este archivo git.php lo subimos a la raiz de nuestro folder (Del repocitorio descargado)

ejemplo :

/public_html/test/NombreDeRepositorio

4) Nos diregimos a la direccion web de nuestro repocitorio en github.

Ejemplo:

https://github.com/NombreDeUsuario/NombreDeRepocitorio

*** Necesitamos ser o tener privilegios de administrador (en nuestra cuenta de GitHub) para poder hacer esto.

Buscamos dentro de la pagina de nuestro repocitorio una opcion llamada Settings
(bloque del lado derecho en la pagina del repocitorio)

Luego de entrar click en "WebHooks & Services"

Click en "Add WebHook"

En el siguiente formulario agregamos

  • Payload URL : http://www.website.com/NombreDelRepocitorio/git.php
    (Agregamos la direccion de nuestro archivo git.php)


  • Payload version: v3+form
    (Dependiendo de nuestra necesidad)




  • Which events would you like to trigger this webhook?:Send me everything.
    (Dependiendo de nuestra necesidad)


    • Para finalizar esto click en "Add WebHook"





    Realizamos cambios al repocitorio de GitHub y luego revisamos en nuestro VPS si los cambios hacen efecto. Con esta configuracion ya deberia estar funcionando.
