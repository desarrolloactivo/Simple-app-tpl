Una vez conectado por medio de ssh en el servidor vps nos dirigimos a la carpeta donde vamos a crear nuestro repocitorio Ejemplo: usuario/public_html/test/ Una vez dentro del directorio vamos a clonar nuestro repositorio github por medio del siguiente comando: git --init git clone git@github.com:NombreDeUsuarui/NombreDeRepositorio.git y con esto se debe crear una carpeta dentro del directorio actual con el nombre de los repositorios luego nos movemos a la carpeta de nuestro repocitorio recien creado (segun nuestro ejemplo seria NombreDeRepocitorio) cd /public_html/test/NombreDeRepositorio Lo siguiente seria hacer un test manual realizando cambios en el repocitorio y luego mediante git pull Revisamos si se actualizaron los archivos del repocitorio en github a nuestro VPS 3) Creamos un archivo php Ejemplo git.php en el vamos a agregar el siguiente codigo
